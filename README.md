### 프로젝트 일정
* 1,2차 개발 기간 : 32일(2022.8.1 ~ 9.1) - 현재 3차 개발 진행중
* 총 개발 시간 : 480H(15H x 32일)
* 1인당 일일 평균 개발 시간 : 15H
* 인원 : 7명
